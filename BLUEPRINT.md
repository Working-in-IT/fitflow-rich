# FitFlow — BLUEPRINT

## Текущее состояние

- **Версия приложения:** 4.2 (релиз Q1 2026).
- **Пользователи:** ~150K активных в месяц (MAU), ~50K в неделю с ≥3 тренировками (WAU3+).
- **Подписки:** ~22% trial→paid conversion (ниже target 25%). Annual:Monthly ratio = 23% (ниже target 30%).

## Roadmap Q2 2026

### Запущено / в работе

- [x] **Beginner Mode (Q1 release).** Новый сегмент контента для новичков с акцентом на правильную технику. Снизил тикеты «слишком сложно» с 18% до 7%.
- [ ] **Mood check-in перед тренировкой** (in progress, релиз mid-Q2). Опросник на 3 вопроса перед началом тренировки — для дополнительной персонализации.

### Бэклог (приоритизирован)

1. **Редизайн onboarding flow** — цель довести trial→paid до 28%. Гипотеза: шаг 2 (выбор целей) — точка прерывания, нужно перенести после первой тренировки.
2. **Персонализация рекомендаций для "Молодой Мамы"** — отдельная карусель beginner-контента на главной для постродовых пользователей.
3. **Annual subscription push** — пуш-кампания для monthly-подписчиков с предложением annual со скидкой.
4. **Социальные функции v0** — friends list + share workout (для персоны «Студент»).
5. **Apple Watch app** — watchOS-приложение с пульсометром.

## Последние решения

- **2026-03:** убрали из onboarding обязательный выбор фитнес-целей на step 2, теперь это опционально. Конверсия онбординга выросла на 4 пункта.
- **2026-02:** запустили beginner-карусель для пользователей, выбравших «новичок» в `onboarding_step_1_profile`.
- **2026-01:** перешли с monthly-only подписки на двойную (monthly + annual).

## Активные исследования

- Анализ feedback по теме «мотивация и срывы» — почему пользователи теряют интерес после 2–3 недель.
- A/B-тест: что эффективнее как retention-механика — streak-счётчик или weekly goal.
