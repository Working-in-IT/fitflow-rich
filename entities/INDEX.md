# Entities Index

Reference catalog of domain entities for FitFlow.

- [user](user.md) — User lifecycle (registered → trial → paid → churned/renewed)
- [workout](workout.md) — Workout types, intensity, properties
- [session](session.md) — Session definition and engagement metrics

## Adding a new entity

Create `entities/<slug>.md` describing the domain object. Add it to this index. Update `KNOWLEDGE_LOG.md` with the decision.
